# Emboitage Brand Images

Please save your brand images here:

1. **emboitage-hero.jpg** - The green image with the contact info and product showcase (left image)
   - This will appear in the hero section of the homepage
   - Recommended size: 800x600px or larger

2. **emboitage-welcome.jpg** - The green image with the cube logo (right image)  
   - This will appear in the welcome/about section
   - Recommended size: 600x600px or larger

## How to add the images:
1. Save both images from the attachments you provided
2. Rename them as described above
3. Place them in this directory: `backend/static/images/`
4. The website will automatically display them on the homepage

Note: Make sure the images are in JPG format with good quality for best results.
